# 🎵 Youme

### _Modern, Fresh, & Lightweight YouTube to MP3 Converter_

**Youme** adalah aplikasi berbasis web yang dirancang untuk mempermudah konversi video YouTube menjadi file MP3 berkualitas tinggi secara instan. Dibangun dengan fokus pada estetika **Glassmorphism** dan efisiensi _backend_, Youme menawarkan pengalaman pengguna yang mulus tanpa beban iklan yang mengganggu.

## ✨ Fitur Utama

- **Glassmorphism UI:** Antarmuka modern dengan efek blur kaca yang estetik dan responsif di semua perangkat.
- **Smart Auto-Cleanup:** Sistem cerdas yang otomatis menghapus file lama dan membatasi penyimpanan maksimal 3 file untuk menjaga efisiensi server.
- **Secure Filename Logic:** Otomatis membersihkan karakter berbahaya (seperti `#`, `&`, atau spasi) agar file aman saat diunduh.
- **High Quality Audio:** Konversi MP3 standar industri hingga 192kbps menggunakan kekuatan FFmpeg.
- **Environment Aware:** Konfigurasi dinamis untuk deteksi lingkungan lokal maupun serverless (seperti Vercel/Render).

---

## 🛠️ Tech Stack

- **Backend:** Python & Flask
- **Core Engine:** [yt-dlp](https://github.com/yt-dlp/yt-dlp)
- **Converter:** FFmpeg
- **Frontend:** HTML5, Modern CSS (Glassmorphism), JavaScript (Fetch API)
- **Icons:** FontAwesome 6
